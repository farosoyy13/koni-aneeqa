# **أناقة CHIC** — منصة الفخامة والجمال

> "حيث تلتقي الرقة بالتميز.. في **أناقة CHIC**، نحن لا نبيع الفساتين، بل نصيغ لحظاتك التي لا تُنسى."

**أناقة CHIC** هي وجهتك الرقمية الأولى لاختيار فساتين السهرة والفساتين الناعمة الأكثر رقياً. صُمم هذا المتجر ليجمع بين تجربة تسوق عالمية الطراز وبين الذوق العربي الرفيع، مع دعم كامل لأحدث تقنيات الويب لضمان سرعة وسلاسة لا تضاهى.

---

## 🚀 لماذا **أناقة CHIC**؟
*   **تصميم ملكي:** واجهة باللغة العربية (RTL) بتدرجات الألوان الذهبية والأسود الفخم.
*   **تجربة تقنية فائقة:** مبني بأحدث إصدارات React و Vite مع Tailwind CSS.
*   **ذكاء بصري:** نظام عداد مشاهدات تفاعلي (Interactive Viewers Counter) لتعزيز الحماس.
*   **دعم المدفوعات الحديثة:** تكامل سلس مع خدمات التقسيط الرائدة (تمارا وتابي).
*   **أداء صاروخي:** سرعة تحميل فائقة وتوافق تام مع أجهزة الجوال.

---

## 🏗️ هيكل المشروع التقني
*   `frontend/` — واجهة المتجر (React 18+, Vite 5, Tailwind CSS 4, Wouter).
*   `backend/` — الخادم الخلفي (Node.js, Express) مع نظام حماية Firebase.
*   `database/` — بنية بيانات مرنة باستخدام Drizzle ORM و PostgreSQL.
*   `api-spec/` — توثيق معماري دقيق (OpenAPI 3.1).
*   `capacitor/` — جسر ربط لتطبيق الهاتف (iOS/Android).

---

## ✨ ميزات حصرية
*   **سلة ذكية:** حفظ تلقائي لمحتويات السلة عبر التخزين المحلي (LocalStorage) المطور.
*   **نظام صلاحيات المالك:** لوحة تحكم سرية ومؤمنة بالكامل لا يراها ولا يصل إليها إلا صاحب المتجر.
*   **تفاعلية عالية:** مكتبة `framer-motion` لانتقالات بصرية ساحرة.
*   **تواصل مباشر:** نظام مراسلة مدمج عبر واتساب وسناب شات للتواصل الفوري.

---

## 🛠️ إعدادات التطوير
1.  **التثبيت:** `npm install` (من المجلد الرئيسي).
2.  **التشغيل:** `npm run dev` (يتم تشغيل الواجهة والخادم بضغطة زر).
3.  **النشر:** نظام نشر آلي جاهز عبر Firebase Hosting.
