# كوني أنيقة — متجر الفساتين الراقية

متجر إلكتروني فاخر للفساتين النسائية الراقية، مبني بـ React + Vite + Node.js + PostgreSQL.

## هيكل المشروع

```
frontend/    — واجهة المتجر (React, Vite, Tailwind CSS, Wouter)
backend/     — الخادم الخلفي (Node.js, Express)
database/    — مخططات قاعدة البيانات (Drizzle ORM, PostgreSQL)
api-spec/    — مواصفة API (OpenAPI 3.1)
capacitor/   — إعدادات تطبيق iOS (Capacitor 6)
```

## الميزات
- تشكيلة فساتين (سهرة / ناعمة) مع صور وأسعار بالريال السعودي
- سلة مشتريات تفاعلية محفوظة في localStorage
- اختيار المقاس واللون لكل فستان
- دفع محاكى (مدى / فيزا)
- آراء وتقييمات العميلات
- نموذج تواصل متكامل
- تصميم RTL عربي كامل — وردي ناعم وذهبي ملكي
- قاعدة بيانات PostgreSQL مع Drizzle ORM
